
# 🚀 ASSEMBLE

A college collaboration platform that helps students discover hackathons, find teammates, and build teams based on skills and interests.

## 💡 Inspiration

ASSEMBLE was inspired by a real problem. During a hackathon registration, one of my friends was left without a team because the maximum team size was reached. Although she knew other classmates, she didn't have the network to quickly find teammates. This sparked the idea of creating a platform where students could connect and collaborate more easily.

## ✨ Features

* 🔍 Browse hackathons
* 📢 Post hackathon opportunities
* 🤝 Apply to join teams
* 🧠 Skill-based match percentage
* 💬 Chat and communication system
* 🔐 Firebase Authentication
* 📊 Application management dashboard

## 🛠️ Tech Stack

**Frontend:** HTML, CSS, JavaScript

**Backend & Database:** Firebase Firestore, Firebase Authentication

## 🎯 Problem Solved

Students often miss opportunities because they don't know the right people. ASSEMBLE helps bridge that gap by connecting students based on skills rather than existing networks.
